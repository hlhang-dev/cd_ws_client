# Changelog

All notable changes to this package will be documented in this file.

## 0.0.1

### Added

* Initial release of `cd_ws_client`.
* Added reusable WebSocket client based on `web_socket_channel`.
* Added connection lifecycle management.
* Added WebSocket status stream.
* Added message broadcast stream.
* Added JSON message parsing.
* Added authentication message support through token config.
* Added heartbeat support.
* Added automatic reconnect support.
* Added configurable reconnect policy.
* Added subscribe and unsubscribe helper methods.
